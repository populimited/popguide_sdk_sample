# Popguide SDK - iOS Sample App

This is the iOS sample app to help you integrating the popguide SDK into your app.

## Documentation

- [Documentation](./PopguideSDK.doccarchive)

## How to import SDK

- Tap on Your project
- Tap on the Target where you want to include the Framework
- Tap on General
- Press the + Button on Frameworks, Libraries, and Embedded Content
- Tap op "Add Other" and "Add Files"
- Search for the "PopguideSDK.xcframework" to include
